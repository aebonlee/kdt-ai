## 4. 유지보수 메모 (중요)

- **강의 데이터 진실 원본은 `src/data/lecture*.js`(모놀리식)** — 여기서 편집한다.
- `scripts/split-lectures.mjs`가 subjectId별 모듈(`src/data/lectures/<id>.js`)을 생성 → **`predev`/`prebuild`로 자동 실행**(생성물은 `.gitignore`). 수동 재생성은 `npm run split`.
- `Lectures.jsx`는 `import.meta.glob`로 현재 과목만 동적 로드(stale 가드·로딩 표시). curriculum·PERIOD_TIMES는 정적.
- 콘텐츠 수정 후 `npm run build`(또는 dev)만 하면 분할이 자동 반영됨.
- **reference/ 원자료**: 실라버스 34종·교재 PDF는 비공개 Google Drive + 데이터드라이브 `/Volumes/aebon - 데이터/.../skala/docs`.
