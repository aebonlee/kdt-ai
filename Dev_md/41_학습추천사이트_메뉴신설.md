# 41. 학습추천사이트 메뉴 신설

**작업일**: 2026-07-16
**요청**: 상단 메뉴 "게시판" 다음에 학습추천사이트 — 대표 운영 학습 사이트 11곳 게시, 1회 로그인 편리함 + AWS 쿠폰 발행 안내 포함

## 구성
- `src/data/recommendsites.js` — 11개 사이트 3그룹 분류 + 안내문 정본
  - **자격증 대비(5)**: aice(KT AICE) · aws(AWS, 🎟️ 응시 쿠폰 발행 안내 배지) · eip(정보처리) · coding(정보처리 실기 C/Java/Python) · linux-study(리눅스 마스터)
  - **프로그래밍·CS 기초(4)**: python-study · java-study · data-structure · algorithm
  - **심화·종합(2)**: bootcamp · statistics
- `src/pages/Recommend.jsx` — 상단 안내 박스 2개(🔑 1회 로그인 계정 공유 / 📌 이용 팁: 과정 학습 우선·귀가 후 단계별 정복), 그룹별 카드 그리드(새 탭 열기), 하단 무료 운영·AWS 쿠폰 각주
- 라우트 `/recommend` (공개 — 로그인 게이트 없음), 헤더 nav "게시판" 다음에 "학습추천사이트" 추가

## 근거 문안
- 대표가 수강생 공지로 공유한 사이트 목록·소개 문구 그대로 반영("디자인보다 콘텐츠 중심, 내용 검증", "매일 집중 학습 우선, 귀가 후 틈틈이")
- 계정 공유 안내: 드림아이티비즈 사이트 공통 회원 풀 — 한 번 가입한 구글·카카오 계정으로 전 사이트 로그인
