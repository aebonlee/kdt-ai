## 2) 모바일 최적화 — 햄버거 네비게이션
상단 메뉴 11개가 모바일에서 flex-wrap으로 지저분하게 늘어지던 문제를 햄버거 드로어로 개선.
- `Header.jsx`: 햄버거 토글 버튼(3-bar→X 애니메이션), 슬라이드다운 드로어, 라우트 이동/항목 클릭/오버레이 클릭 시 자동 닫힘, 열림 시 body 스크롤 잠금, `aria-expanded`·`aria-controls` 접근성.
- `index.css`: `≤880px`에서 가로 메뉴 → 절대배치 드로어(세로 스택, 활성 항목 좌측 골드바), 다크모드 대응. `≤600px`에서 brand-sub/sep 숨김·로고 축소.

## 3) 4기 공식 실라버스 대조검증
`/Volumes/aebon - 데이터/skala-4` 의 4기 Syllabus(PPTX 15종)·실습 가이드(xlsx)를 추출해 담당 15개 과목 전부를 현재 강의안과 대조.
- **결과: 학습목표·주요 학습내용·종합실습이 공식 실라버스와 정확히 일치** — 기존 콘텐츠가 실라버스 그라운딩으로 제작된 덕분. 대표 확증 사례:
  - langchain Day2 종합실습 "댓글 요약·감정·키워드 병렬 → 부정 분기" = 사이트 langchain-2 예제와 동일
  - serving Day2 "FastAPI Dockerize + model_version" = 사이트 serving-2 Dockerfile/메트릭 예제와 부합
  - python Day1·2 교시별 실습 = 사이트 python-1·2 "교시 시연" 예제와 매핑
  - transformer 종합실습 "PyTorch/Colab 미니 언어모델" = 사이트 transformer 미니모델 예제와 부합
- 실습 가이드 xlsx는 평가기준·제출물·점수분포 운영지침(코드 아님) — 콘텐츠 사실관계에 영향 없음.
- 결론: 사실관계 수정 불필요, 깊이 보강(1항)이 이번 실질 개선.
