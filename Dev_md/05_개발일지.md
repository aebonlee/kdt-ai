# 05. 개발일지 (Development Journal)

SKALA 4기 강의 안내 사이트 개발 기록. (개발 기간: 2026-06-10 ~ 2026-06-11)

---

## 개요
- **프로젝트**: SKALA(SK AI Leader Academy) 4기 — 이애본 강사 담당 강의 안내 사이트
- **목표**: 담당 강의의 과목·일자별 학습내용을 학습자에게 제공
- **레포/배포**: `aebonlee/skala` → https://skala.dreamitbiz.com (GitHub Pages, Actions 자동배포)

## 일자별 진행

### 1) 초기 셋업 & 디자인 확립
- 빈 저장소(CNAME만 존재)에서 React/Vite 골격 구축
- 디자인 방향 전환: 초기 Tailwind → `data.dreamitbiz.com`의 **Editorial Navy** 레이아웃 이식 → **SKALA 공식 브랜드 컬러**(딥 인디고 #0E1152 / 액센트 #3F51FF) 적용
- 라이트(기본) + 다크 모드 토글, SKALA 로고·OG 이미지 적용
- 공식 사이트의 **커리큘럼 Overview**(7~12월, 평일 09:00~18:00, 100% 오프라인) 반영

### 2) 확정 일정 반영 (핵심 데이터)
- 산정 규칙 확정: 확정표에서 **"확정" + 비고(기타) 열 동그라미(o) = 담당 강의일**
- 1차 37일 → 일정 변경 반영 후 **최종 32일 / 11과목**
- **광주 별도 분반** 구분(판교4반·3반·광주1반, 색 구분)
- 일정-강의안 정합성 자동 검증: 요일·지역·분반·차수·강의안 매칭 전수 확인 (RAG 09/21 → Day3 보정)

### 3) 메뉴/페이지 확장
- 상단 메뉴 다단계 확장: About·수업일정표·선수학습자료·과목별 안내·강의안·팀 프로젝트·참고자료·학습관리·게시판·학습자료실·대시보드
- **선수학습자료**: 17개 주제 → 카테고리 탭 + 주제별 개별 페이지(심화/실수/체크리스트)

- **팀 프로젝트**: 11단계 가이드
- **학습관리**: 32일 자가평가 진도율(localStorage)
- **수업일정표**: 7~10월 달력 + 타임라인

### 4) Supabase 연동
- 구글/카카오 OAuth 로그인, 게시판(Q&A·공지·댓글), 대시보드(학습자/강사) 스캐폴드
- 빌드 env(GitHub Actions secret) 주입, `supabase/schema.sql`(RLS) 제공

### 5) 콘텐츠 심화 (반복 보강)

- 코드: 주석 **녹색 표기** + **복사 버튼** 전 페이지 적용
- 교시 시간표 정확화(1~8교시, 09:00~17:50)

## 주요 기술 이슈 & 해결
| 이슈 | 원인 | 해결 |
|------|------|------|
| git rebase 반복 실패 | macOS 한글 파일명 NFC/NFD 정규화 불일치 | `core.precomposeunicode` + `merge` 사용 |
| 새 Pages가 legacy로 시작 | GitHub Pages 기본 build_type | `gh api`로 workflow 전환 + 커스텀 도메인 설정 |
| 새로고침 시 404 | SPA + Pages에 실파일 없음 | `404.html` 폴백 + index 경로 복원 스크립트 |
| 파비콘 미표시 | 받아온 favicon.ico가 실제 HTML | SKALA 브랜드 SVG/PNG 파비콘 신규 생성 |
| 좌측 메뉴가 길어 잘려 보임 | sticky 사이드바 스크롤바 숨김(macOS) | 스크롤바 상시 표시 + 월/카테고리 탭 도입 |
