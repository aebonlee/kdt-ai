## 2. 소속 분반 온보딩 (ClassOnboarding)
- 로그인 후 프로필 미완성 시 **모달 자동 표시**: 역할(학생/교수자) → 학생은 캠퍼스·반(호실 표기) 선택, 교수자는 담당 분반 복수 선택 → **"맞음을 확인합니다" 체크** 후 저장(확인 체크 없으면 저장 불가)
- "나중에"는 세션당 1회 스누즈(sessionStorage) — 다음 접속 시 다시 표시(반복 확인)
- **주기 재확인**: confirmed_at이 14일 지나면 상단 배너 "소속 정보 확인 — ○○이 지금도 맞나요? [✓ 맞아요] [변경할래요]" — 반복 확인으로 정보 정확도 유지(RECONFIRM_DAYS=14, useProfile.js)
- 상단바 로그인 상태 옆에 소속 배지 표기: 학생 "판교 5층 7반" / 교수자 "담당 N개반"
- 분반 정의: `src/data/classes.js` — 18개반(반별 시간표_F 기준 호실 포함)

## 3. 학습관리 — 분반 수강 리스트 체크
- 학생(소속 등록 완료): **우리 반이 실제 수강하는 날짜 리스트**(담당 세션 + othersessions의 해당 트랙 셀 병합, 월별 그룹)로 체크
  - 담당 강의=골드 "이애본 강사", 타 강사=다크그린 제목+강사명, 특강·행사는 체크 제외(· 표시)
  - 각 항목 "학습 →" 링크: 담당→강의안 날짜, 타강사→etc/참고자료 강의안
  - 진도율 분모 = 우리 반 체크 가능 일수. "소속 변경" 버튼으로 모달 재오픈
- 미등록·교수자·관리자: 기존 담당 과목별 보기 유지 + "소속 등록하기" 안내
