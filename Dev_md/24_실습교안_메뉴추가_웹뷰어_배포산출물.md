# 24. 실습교안 메뉴 추가 · 실습 교재 웹 뷰어 · 배포 산출물

## 개요
실습예제 보강분과 4기 실라버스 그라운딩을 바탕으로 **담당 18과목 실습 교재**를 산출물(PDF·웹)로 만들고, 사이트 상단 메뉴를 정리했다.

## 1) 상단 메뉴 개편 (Header.jsx)
- **삭제**: `학습자료실`(padlet 외부링크)
- **변경**: `강의안` → `학습강의안`
- **추가**: `실습교안`(학습강의안 옆) → `/practice-textbook.html` 새 탭

최종 순서: About · 수업일정표 · 선수학습자료 · 과목별 안내 · **학습강의안** · **실습교안** · 팀 프로젝트 · 참고자료 · 학습관리 · 게시판 · 대시보드

## 2) 실습 교재 생성기 (scripts/build-textbook.mjs)
커리큘럼·강의안 데이터(split 병합본)를 조판해 3가지 산출물 생성:
- **PDF 교재** (`node build-textbook.mjs` → Chrome 헤드리스): A4 403p, `dist-textbook/SKALA_4기_실습교재_이애본.pdf`
- **Artifact 조각** (`--web`): `dist-textbook/textbook-web.html` (claude.ai 뷰어 발행용)
- **사이트 정적 페이지** (`--web`): `public/practice-textbook.html` ("실습교안" 메뉴 대상)

수록 섹션: 표지·목차·학습목표·핵심개념·심화이론·상세학습·진행 시간표·실습·추가 Lab·실습 예제·실전 소스·과제.

### 웹 뷰어 특징
- SKALA 브랜드 팔레트(딥 인디고·액센트·골드), **라이트/다크 테마** 자동 대응
- 좌측 과목 목차(앵커) 스티키 사이드바 + 반응형(≤820px 상단 칩 목차)
- **코드 블록 복사 버튼**(clipboard API + execCommand fallback), 인쇄본은 숨김
- 명칭 "실습 교재", 표기 "담당 18과목"

## 3) 빌드 자동화 (package.json)
`predev`/`prebuild` 에 `build-textbook.mjs --web` 추가 → `public/practice-textbook.html` 을 빌드마다 자동 생성(최신 강의안 반영). 생성물이므로 `.gitignore`(`public/practice-textbook.html`, `dist-textbook/`).

수동 재생성:
- `npm run textbook` → PDF
- `npm run textbook:web` → 웹 뷰어(조각 + 사이트 페이지)

## 버그 수정 — 칩 크기 오류
웹 뷰어에서 칩(배지)이 세로로 늘어난 큰 타원으로 렌더되던 문제: flex 컨테이너 기본 `align-items:stretch`가 원인. 컨테이너 `align-items:center` + 칩 `display:inline-flex`로 정상 알약 복구.

## 검증
- `npm run build` ✅ — prebuild가 practice-textbook.html 생성 → vite가 dist로 복사 확인
- 웹 뷰어 렌더(칩·복사버튼·명칭) 스크린샷 확인
- 화면·모바일·복사버튼 클릭은 대표님이 직접 (skala.dreamitbiz.com 자동배포 후)
