# 04. 종합 정리 (개발 종료)

SKALA 4기 — 이애본 강사 담당 강의 안내 사이트. 1차 개발 종료 시점 종합 정리.

- 배포: https://skala.dreamitbiz.com (GitHub Pages, `main` 푸시 → Actions 자동배포)
- 스택: React 18 · Vite 5 · React Router 6 · Supabase(옵션) · 단일 CSS 디자인시스템(Editorial Navy + SKALA 브랜드)

## 상단 메뉴
About · 수업일정표 · 선수학습자료 · 과목별 안내 · 강의안 · 팀 프로젝트 · 참고자료 · 학습관리 · 게시판 · 학습자료실(패들렛, 새탭) · 대시보드

## 핵심 데이터
- **담당 강의 32일 / 11과목** (확정표의 "확정 + 비고 o" 기준). 광주는 별도 분반.
  - 분반 색: 판교4반=인디고 `#3F51FF` · 판교3반=틸 `#0D9488` · 광주1반=앰버 `#E0892F`
- `src/data/`:
  - `curriculum.js` — 과목·세션(32일)·참고과목(Spring AI)
  - `lectureplans.js` / `lectureperiods.js` — 강의안 8교시 시간표(09:00~17:50)
  - `lectureconcepts.js` · `lecturedetails.js` · `lecturetheory.js` · `lecturetheory2.js` — 핵심개념·상세내용·심화이론·실전소스
  - `resources.js` · `prepdeep.js` — 선수학습 17개 주제(+심화)
  - `teamproject.js` — 팀 프로젝트 11단계 가이드

## 페이지별 구성
- **수업일정표**: 7~10월 달력 + 월별 타임라인, 지역 필터
- **선수학습자료**: 카테고리 탭(전체+9분류) → 주제별 개별 페이지(/prep/:id), 심화·실전·실수·체크리스트
- **강의안**(/lectures): 월 탭(7·8·9·참고·10월 판교·10월 광주). 차수별 = 학습목표→핵심개념→심화이론→상세내용→8교시 시간표→실습→추가Lab→실습예제→실전소스→과제
- **팀 프로젝트**(/team): 개요~회고 11단계
- **학습관리**(/progress): 32일 자가평가 진도율(localStorage), DayDetail 토글 연동
- **게시판·대시보드**: Supabase 로그인(구글/카카오) 필요
