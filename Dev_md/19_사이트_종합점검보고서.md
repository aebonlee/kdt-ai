## 6. 성능 점검

- **최대 청크**: `Lectures` **415.69 kB / gzip 245.79 kB** (콘텐츠 보강으로 증가) — 사실상 전량이 강의 데이터
- **적용 완료(퀵윈)**: Pretendard 통짜(~1MB+)→dynamic-subset+preconnect(FCP↓) · Home 정적 import(워터폴 -1 RTT) · curriculum_graph lazy+크기(CLS·-77KB 지연) · CodeBlock memo · `--azure` 색버그 · 미사용 이미지 삭제
- **대기(대형)**: 강의 데이터를 과목별 동적 로드로 분할 → 첫 진입 **-190KB 이상**(gzip 245→과목당 수~수십KB). 콘텐츠 통합이 끝났으므로 **지금이 착수 적기**
- **npm 취약점**: esbuild(moderate)·vite(high) 2건 — **둘 다 dev 서버 전용**, 정적 배포 프로덕션엔 영향 없음(수정 시 vite 8 breaking이라 보류)

## 7. 기능 스모크

- 라우팅 18개 정상, 알 수 없는 경로 → 홈 정규화
