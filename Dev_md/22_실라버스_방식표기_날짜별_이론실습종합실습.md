# 실라버스 방식(이론/실습/종합실습) 날짜별 강의안 기재

- **대상**: github.com/aebonlee/skala → skala.dreamitbiz.com
- **작업일**: 2026-07-05

## 1. 요청

로컬 폴더의 **실라버스(Syllabus)** 를 확인해, 각 날짜별로 **실습 / 종합실습** 으로 표기된
부분을 날짜별 강의안에 명확히 기재.

## 2. 근거 원본 (로컬 데이터드라이브)

- `/Volumes/aebon - 데이터/SKALA_4/0. Syllabus-…zip` — **4기 Syllabus 34종 PPTX**
- 각 PPTX 의 「기간·목차·주요 학습내용·시간·**방식**」 표에서 **방식** 열을 추출.
  - 방식 값: `이론`, `이론+실습`, `실습`, `개념 실습`, `종합실습`
  - 교육 방식 비율(예: 이론 70% · 실습 30%)은 각 과목 표지의 「교육 방식」에서 확인.
- (참고) `reference/excel/…확인표.xlsx` = 강사·시수표(과목 목록), 날짜별 방식 표기는 없음 → Syllabus PPTX 가 유일한 방식 근거.

## 3. 구현

- **신규 데이터**: `src/data/lecturemodes.js`
  - `modes["<subjectId>-<day>"] = { tag, ratio, note }`
  - `tag`: 그 날 대표 방식 배지 (`이론`/`이론+실습`/`실습`/`종합실습`)
  - `ratio`: 실라버스 교육방식 비율, `note`: 실습·종합실습 명시 구간 설명
  - 18과목 42개 Day 전부 매핑 (git~miniproject).
- **렌더링**:
  - `src/pages/Lectures.jsx` (강의안) — 메타 배지 + 방식 안내 박스(`.mode-note`).
  - `src/pages/DayDetail.jsx` (날짜별 상세) — 동일 배지 + 안내 박스.
  - `modeOf(subjectId, day)` 로 조회.
- **스타일**: `src/index.css` — `.chip-mode`, `.mode-note`, 방식별 색상
  (`이론`=슬레이트, `이론+실습`=퍼플, `실습`=틸, `종합실습`=시그니처 블루).
